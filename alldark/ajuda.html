Gerador de Provas - Ajuda

Sobre

O Gerador de Provas é uma aplicação web feita como projeto para PIBIC-EM/Computação (Uni-FACEF) por Arthur Carrenho. As seguintes libs foram utilizadas:

  • xmlToJson - por metatribal
  • jQuery - por John Resig
  • Imgur-Upload - por carry0987

Tela Inicial

Na tela inicial, as seguintes entradas são apresentadas:

  • Quantidade de questões
  • Quantidade de alternativas
  • Variedade de provas
  • Questões extras

Além de atalhos para utilização do banco de questões

Quantidade de questões

Neste campo, você insere a quantidade de questões desejada para a prova, note que este campo apenas limita a quantidade de questões totais da prova, a quantidade de questões que serão "cadastradas" são a soma da Quantidade de questões com Questões extras.

Quantidade de alternativas

Neste campo, você insere a quantidade de alternativas (como A, B, C, D, E) desejadas para prova.

Variedade de provas

Neste campo, você insere a quantidade de provas diferentes que serão geradas, note que as questões são totalmente sorteadas, mesmo com apenas uma variação de prova.

Questões extras

Neste campo, você insere a quantidade de questões extras a serem "cadastradas", note que estas questões não interferem na quantidade de questões resultantes pelo gerador, apenas insere as questões no sorteio das variações.

Midia e Cabeçalho

Durante o cadastro de questões e alternativas, é possivel inserir um cabeçalho para prova (em formato de imagem) e midia para acompanhar as questões.

Insira no campo um link direto para imagem. O Gerador de Provas acompanha um atalho para envio de imagens com link direto (Enviar imagens).

Enviar imagens

Utilize essa pagina para enviar imagens e gerar links diretos para serem utilizados nos campos midia e cabeçalho. Arraste a imagem para a zona pontilhada ou clique nela para procurar arquivo para enviar as fotos, após isto, o link sera apresentado.

Banco de questões

Na área importar do campo de Banco de questões você pode utilizar o arquivo gerado pela área exportar. Você também pode utilizar arquivos xml gerados pelo Excel, utilizando este modelo.

Trabalhando com Excel

Você pode nativamente abrir os arquivos XML gerados pela exportação do banco de questões, note que ao salvar, talvez o programa não retorne como padrão a extenção de origem (XML).

Para salvar corretamente o arquivo, seja pela importação de XML ou utilizando o modelo de geração, salve como Dados XML (*.xml). Tome cuidado para não confundir com Planilhas XML ou Planilhas Excel (.xlsx) (Opção padrão da janela de salvamento).

Modo de importação

Ao importar o arquivo, você pode escolher entre o modo linear e o modo aleatório.

Modo Linear: As questões serão importadas na ordem do banco de questões

Modo Aleatorio: As questões serão sorteadas dentro do banco de questões

Note que não é possivel selecionar um numero de questões maior que o numero de questões do banco de questões.
